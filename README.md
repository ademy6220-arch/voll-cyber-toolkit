# 🔍 LUXA - Open Source Research Toolkit

LUXA is a simple, modular and beginner-friendly OSINT-style research toolkit written in Python.  
It is designed for learning, automation practice, and basic information gathering from public sources.

---

## ⚙️ Features

- 📝 Notes system (save & manage local notes)
- 🎯 Target manager (organize research targets)
- 📊 Statistics overview
- 📤 Export reports to `.txt`
- 💾 Local JSON storage
- 🧩 Clean modular structure
